3 / 14

[API 및 서비스 사용 설정]을 클릭해주세요.
5 / 14
API 목록이 나오면 "YouTube Data API v3"을 찾아 클릭해주세요.
6 / 14

7 / 14
API가 활성화되었으면 API 키를 발급받아야합니다.
[사용자 인증 정보 만들기]를 클릭해주세요.
8 / 14

10 / 14
생성된 API 키를 아래 빈칸에 입력해주세요. (API 키 우측의 아이콘을 클릭하면 복사됩니다.)

끝!

원하는 plug.dj 페이지에 접속한 후, 추가한 플디픽스 북마크를 클릭하면 픽스가 적용되었음을 알리는 창이 나타납니다.
픽스가 적용되었다면 유튜브 검색이 정상 작동하는지 확인해보세요. (페이지를 새로고침하면 픽스도 다시 적용해야합니다.)
